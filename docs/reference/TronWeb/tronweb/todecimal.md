Usage

```javascript
tronWeb.toDecimal()
```

**Parameters**\
String-Hex string

**Returns**\
Number-The hexadecimal value represented by the passed in string.

**Example** 

```javascript
tronWeb.toDecimal('0x15')
>21
```
