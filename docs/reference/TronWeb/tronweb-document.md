[TronWeb](https://tronweb.network/) aims to deliver a unified, seamless development experience influenced by Ethereum's [Web3](https://github.com/web3/web3.js) implementation. We have taken the core ideas and expanded upon them to unlock the functionality of TRON's unique feature set along with offering new tools for integrating DApps in the browser, Node.js and IoT devices. For detailed information please refer to [TronWeb Document](https://tronweb.network/docu/docs/intro/).

You can get all APIs provided by TronWeb at [here](https://tronweb.network/docu/docs/API%20List/).
