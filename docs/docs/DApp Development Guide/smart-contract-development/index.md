This tutorial guides the user on creating, compiling, deploying, and interacting with the Hello World smart contract on &lt;TRON&gt; using [Tron-IDE](http://www.tronide.io). This tutorial requires some familiarity with [Solidity](https://en.wikipedia.org/wiki/Solidity). As a result of the walk-through, the user will become familiar with the process of creating, compiling, deploying, and interacting with smart contracts on the Tron blockchain.
