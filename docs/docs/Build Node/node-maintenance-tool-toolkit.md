Toolkit is a utility suite for java-tron node maintenance, aiming to simplify various development tasks. The current release supports the following features, with more to come:

* [Database Partition Tool](https://tronprotocol.github.io/documentation-en/using_javatron/toolkit/#database-partitioning-tool)
* [Lite Fullnode Data Pruning](https://tronprotocol.github.io/documentation-en/using_javatron/toolkit/#lite-fullnode-data-pruning)
* [Data Copy](https://tronprotocol.github.io/documentation-en/using_javatron/toolkit/#fast-data-copy-tool)
* [Data Conversion](https://tronprotocol.github.io/documentation-en/using_javatron/toolkit/#data-conversion-tool)
* [LevelDB Startup Optimization](https://tronprotocol.github.io/documentation-en/using_javatron/toolkit/#leveldb-startup-optimization-tool)

<br />

For a detailed introduction to obtaining and using the Toolkit, please refer to the [Toolkit User Guide](https://tronprotocol.github.io/documentation-en/using_javatron/toolkit/).
